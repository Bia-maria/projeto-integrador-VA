1. Introdução
Este projeto apresenta um aplicativo simples para cadastro e organização de intenções de missa. Ele foi desenvolvido em parceria com a Paróquia Nossa Senhora Aparecida de Goianápolis–GO, pertencente à Diocese de Anápolis. O objetivo é substituir o registro manual por um sistema digital rápido, organizado e fácil de usar.

2. Desenvolvimento
O aplicativo foi feito no Android Studio, usando Kotlin, SQLite e telas em XML com ViewBinding. Foram criadas funções básicas, como:
Cadastrar intenções de missa;
Listar intenções registradas;
Editar informações;
Excluir quando necessário.

A estrutura principal inclui:
MainActivity.kt – tela de cadastro;
ListaIntencoesActivity.kt – lista de intenções;
DatabaseHelper.kt – banco de dados;
Intencao.kt – modelo dos dados.

3. Resultados
O aplicativo funcionou bem nos testes, permitindo cadastrar, consultar e editar intenções com facilidade. A secretaria da paróquia avaliou positivamente pela praticidade e organização.



O projeto ajudou a modernizar o trabalho da secretaria paroquial, trazendo agilidade e organização. Também proporcionou aprendizado prático sobre desenvolvimento Android, banco de dados e boas práticas de programação.
